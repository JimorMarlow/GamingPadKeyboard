# GamingPadKeyboard

Разработка игровой механической клавиатуры для игр (в основном для шутеров)

## Требования:
- Расширенная версия механической клавиатуры [Zuoya GMK26](https://ali.click/tiospf)
- Ortholinear (ортолинейное расположение клавиш)
- QMK/VIA (с программируемой RGB LED подстветкой с южным мостом)
- Gasket 
- Bluetooth 5.0/2.4ghz/Wired
- Hot Swappable 
- Programmable for Win/Mac
- [Galaxy Keycap MOA](https://ali.click/6jqspv)
- Silent свитчи (комбинированные: линейные на перемещения и модификаторы SHIFT, ALT, CTRL; тактильные (тугие, 65+g) на кнопки взаимодействия и переключения. 
- FN кнопка для переключения между слоями для Battlefield (пешком, транспорт, вертлет etc.)

## Дизайн
Для работы я использую Zuoya GMK87 (TKL) клавиатура и расширенный цифровой нумпад [Zuoya GMK26](https://ali.click/tiospf) для игр. Мышку я держу левой рукой, а играю правой. Поэтму большинстов игровный одноручных клавиатур мне не очень подходит в основном из-за SPACE, который обычно нажимается левым большим пальцем. Вдохновением является механическая ортолинейная механическая клавиатура [Iris](https://keeb.io/collections/iris-split-ergonomic-keyboard?srsltid=AfmBOooUTG2XE8N4FeXTsp5jLS97ghhxMDJKFsGKGj0ufurZ3tln2wmG), но мне нужен фактически зеркальный вариант правой клавиатуры с кнопками от левого. И мне не нужны эргономичные кнопки под большой палец, меня вполне устраивает ортолинейно расположенный SPACE вместо широкого "0" на цифровом нумпаде, как это сделано в [Zuoya GMK26](https://ali.click/tiospf). Единственный недостаток GMK26 - это нехватка примерно 5-10 клавиш для игр FPS (first person shooter) типа Battlefiels, Escape From Tarkov etc, где используется много дополнительных клавиш. Поэтому я хочу добавить к GMK26 еще два ряда кнопок + еще один энкодер (верхний для громкости мастер звука, + дополнительный)

Выбор кейкапов Galaxy MOA обусловлен следующими факторами:
- они мне нравятся (и используются в игровой связке на GMK87 + GMK26)
- MOA профиль удобень для игр, кнопки одинаковые по высое и взаимозаменяемы в раскладке. Я могу любую клавишу расположить в любом ряду, что не получится в форм-фактре Cherry

## Этапы разработки
- [ ] Создать раскладку в [Keyboard Layout Editor](https://www.keyboard-layout-editor.com/)
- [ ] Изучить основы KiCad и ZMK (гайды ai03 по PCB)
- [ ] Спроектировать схему в KiCad. Макет, матрица, периферия (USB, энкодер, RGB)
- [ ] Развести PCB
- [ ] Заказать производство PCB и компонентов (JLCPCB, LCSC)
- [ ] Напиасть конфиг для ZMK (матрица, пины, базовая раскладка)
- [ ] Пайка платы
- [ ] Спроектировать корпус, ориентируясь на дизайн Zouya
- [ ] Заказать производство корпуса (ЧПУ) или сделать самому на 3D-печати
- [ ] Подобрать стабилизаторы 2U - 2шт
- [ ] Сборка, тестирование

## Прошивка (Firmware) с поддержкой VIA
Основной упор на беспроводное использование. Нужен аккумулятор повыешнной ёмкости (4000-8000 mА/h).
Похоже нужно ориентироваться на nRF52840 + VIA. На первый взгляд, для беспроводной клавиатуры подходит проект ZMK (Zephyr Mechanical Keyboard)
