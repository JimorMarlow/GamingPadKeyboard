### Разработка дизайна раскладки
Промт для perplexity:

Проект кастомной клавиатуры для игры одной правой рукой.
Нужно создать проект раскладки клавишь в Keyboard Layout Editor. 
- За основу взять дизайн Zuoya GMK26
- сдвинуть ряд с энкодером и M1-M5 влево на одну позицию
- вместо этого ряда вставить 6 вертикальных кнопок сверху-вниз: ESC, ~, Tab, CapsLock, Shift, Ctrl
- широкий плюс на цифровой клавиатуре разбить на две одиночные кноки (сверху R, снизу F)
- добавить справа еще ряд из 6 кнопок с отступом в 0.25 (сверху-вниз 0, 5, T, G, B, V)
- M5 заменить на FN
- над блоком цифровых клавиш должны быть цифры по рядам, верхний 6, 7, 8, 9, ниже 1, 2, 3, 4
- цифровые клавиши заменить на QWE, ASD, ZXC. Нижний широкий 0 заменить на SPACE
- второй энкодер снизу на боковой поверхности для управления большим пальцем правой руки
